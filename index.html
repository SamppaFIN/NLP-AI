⚙️ Asetukset

OpenRouter API -avain

Liitä oma OpenRouter API-avain käyttääksesi AI-terapiaa. Avain tallennetaan vain omaan selaimeesi (localStorage). Ilman avainta käytetään sisäänrakennettuja demovastauksia.

AI-malli (valinnainen)

⚠️ API-avaimesi näkyy selaimen kehittäjätyökaluissa (Network-välilehti). Tämä sovellus on tarkoitettu henkilökohtaiseen käyttöön.

ℹ️ Mikä on NLP ja tietojen käsittely

Mikä on NLP?

NLP (Natural Language Processing) on tekoälyn osa-alue, joka mahdollistaa tietokoneiden ymmärtää ihmisten kieltä. TerapiaAI:ssa:

  • Ymmärretään vastauksesi ja tunnistetaan tärkeimmät teemat
  • Generoidaan henkilökohtaisia terapeuttisia vastauksia
  • Luodaan sessioiden yhteenvetoja

Miten tietojasi käsitellään?

  • 🔒 Kaikki keskustelut pysyvät omassa selaimessasi
  • 📋 Keskusteluhistoria tallennetaan localStorage-muistiin
  • 🔑 API-avain on vain sinun hallussasi
  • 🤖 AI-kutsut tehdään salatusti HTTPS:n kautta OpenRouteriin
  • 🗑️ Voit tyhjentää avaimen milloin tahansa asetuksista

💡 TerapiaAI on apuväline — ei korvaa ammattimaista terapiaa. Jos tarvitset kriisitukea, ota yhteyttä ammattilaisiin.
